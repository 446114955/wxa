# @wxa/plugin-replace

实际开发项目中，我们往往需要区分测试和生产环境参数，针对不同环境做不同处理，譬如测试环境下允许debug，生产环境又严格禁止。

使用plugin-replace可以很方便的替换js, json, template, css中匹配的字符串🎉。

# 安装

# 使用npm安装
npm i -S @wxa/plugin-replace
1
2

# 用例

// wxa.config.js
module.exports = {
    plugins: [
        // 传递一个规则数组
        new ReplacePlugin({
          list: [{
            regular: new RegExp('APP_ENV', 'gm'),
            value: 'bcds'
          }]
        })
        // 或者一个规则对象，key为目标字符串，value为替换内容
        new ReplacePlugin({
          list: {
            'APP_ENV': 'bcds'
          }
        })
    ]
}
